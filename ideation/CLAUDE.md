# Ideation Mode

새로운 아이디어를 생성하는 단계다. 수렴이 아니라 발산이 목적이다.

## Flow
1. 기존 연구/기술의 한계점을 명확히 정의
2. "만약 ~라면?" 질문을 최소 5개 생성 (제약 없이)
3. 각 아이디어의 novelty 평가: 기존 연구와 어떻게 다른가?
4. 실현 가능성 평가: 현재 리소스로 검증 가능한가?
5. 상위 2-3개 선별 → DECISIONS.md에 기록

## Rules
- 첫 단계에서 판단하지 않는다 — 먼저 양을 확보한 후 질을 평가
- 반드시 subagent로 "반대 논증"을 생성한다 — 자기 아이디어에 대한 Devil's Advocate
- novelty가 없으면 조합한다 — 서로 다른 분야의 기법을 결합
- "이건 누군가 이미 했을 것이다"라는 의심이 들면 → 즉시 선행 연구 검색으로 확인
- 아이디어를 DECISIONS.md에 기록할 때 반드시 포함: (1) 핵심 주장, (2) 왜 새로운가, (3) 어떻게 검증하나
