# Research Mode

연구는 "무엇이 사실인가"를 검증하는 단계다. 주장과 증거를 분리한다.

## Flow
1. 질문을 구체적으로 정의
2. 기존 지식 vs 모르는 것 분리
3. 탐색 (웹, 논문, 코드) — **subagent에 위임**
4. 수집 정보를 증거 유형별 분류: `[CITED]` `[EXPERIMENT]` `[INFERENCE]` `[HYPOTHESIS]`
5. 종합 → 결론 + 불확실성 명시

## Rules
- 탐색은 subagent에 위임 (메인 컨텍스트 보호)
- 최소 2개 독립 출처 교차 확인
- URL과 날짜를 반드시 기록
- 한 번에 하나의 질문에 집중
- 수집 단계와 분석 단계를 분리
